04

各平台安装 & 使用方式

🪟 Windows
01
下载 safeskill-windows.exe
将文件保存到任意目录，例如 C:\Tools\safeskill.exe
02
（可选）加入系统 PATH
将 C:\Tools 加入系统环境变量 Path，之后在任意命令行窗口直接输入 safeskill 即可。
💡
不加 PATH 也没关系，在 .exe 所在目录打开命令行，用 .\safeskill.exe 运行即可。
03
运行
cmd.exe
C:\Tools> safeskill.exe --version
safeskill 1.1.0
C:\Tools> safeskill.exe login --token YOUR_TOKEN
✅ Token 已保存
C:\Tools> safeskill.exe scan my-skill.zip
ℹ 上传文件扫描: my-skill.zip (0.5MB)
✅ 任务已提交
Task ID: a1b2c3d4e5f6...
🍎 macOS
01
下载并挂载 safeskill-macos.dmg
双击 .dmg 文件，弹出磁盘映像窗口
02
将二进制复制到 PATH
Terminal
# 将 safeskill 复制到系统路径
sudo cp /Volumes/SafeSkill\ CLI/safeskill /usr/local/bin/
sudo chmod +x /usr/local/bin/safeskill
# 如果提示"无法验证开发者"，执行以下命令解除限制
xattr -dr com.apple.quarantine /usr/local/bin/safeskill
⚠
macOS 会对未签名程序弹出安全警告。在系统偏好设置 → 安全性与隐私 → 点击「仍然允许」，或用上面的 xattr 命令一劳永逸。
03
运行
Terminal — zsh
~ $ safeskill --version
safeskill 1.1.0
~ $ safeskill login --token YOUR_TOKEN
✅ Token 已保存到 ~/.config/safeskill/config.yaml
✅ Token 验证通过
🐧 Linux
01
下载 safeskill-linux，赋予执行权限
bash
chmod +x safeskill-linux
./safeskill-linux --version
02
（推荐）安装到系统，全局可用
bash
sudo mv safeskill-linux /usr/local/bin/safeskill
safeskill --version # 任意目录可用
03
运行
bash
$ safeskill --version
safeskill 1.1.0
$ safeskill login --token YOUR_TOKEN
✅ Token 已保存
$ safeskill scan-url https://github.com/owner/repo
ℹ 投递 URL 扫描: https://github.com/owner/repo
✅ 任务已提交
05

命令速查表

命令 说明 示例
login 设置 API Token（首次必须执行） safeskill login --token YOUR_TOKEN
whoami 验证 Token 是否有效 safeskill whoami
scan 上传本地 zip 文件扫描 safeskill scan my-skill.zip
scan-url 投递 GitHub/URL 扫描 safeskill scan-url https://github.com/x/y
report 查询扫描报告 safeskill report <task_id> --wait
report-batch 批量查询多个报告 safeskill report-batch id1 id2 id3
judge 通过 MD5/SHA1 研判查询 safeskill judge --md5 a5d5c01672e...
search 按 Skill 名称搜索 safeskill search tavily-search
download 下载资源包 safeskill download --task-id <id>
logout 清除已存储的 Token safeskill logout
config set 修改配置项 safeskill config set output.format json
💡
常用参数：
--output json 输出 JSON 格式  |  --wait 等待扫描完成再返回  |  --verbose 显示详细信息  |  --no-color 禁用彩色输出（适合日志）
